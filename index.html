CSS Properties: A Cheat Sheet

CSS Properties are what we use to modify HTML elements. A number of these properties can be found in the table below, along with some examples.

Property Example Explanation
background-color background-color: deepskyblue; This property modifies the background color of the element it is applied to. In the example code, the background color of the element would be set to deepskyblue.
border border: 1px solid black This property applies a border round the specified element. It has been used right here, in this table, to create the lines around all of the elements in this table. This specific example would apply a border which is a solid line, one pixel wide, and black in colour, around the elements it is applied to.
color color: orange; This property modifies the actual color of the element it is applied to. For example, if it was applied to a <p> element, the text would appear like this.
font-family font-family: Helvetica; This property modifies the font the selected element appears in. The text on this page is in Helvetica, while the elements are in monospace.
font-weight font-weight: bold; This property changes whether an element is shown in bold. Bold is a binary setting, but numbers can also be used, such as 400, 600, or 900.
text-align text-align: center This property modifies the alignment of the element. This paragraph is, by default, aligned to the left, whereas the title is aligned to the center.
